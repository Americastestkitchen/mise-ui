## Line Height Suggestions
### Explicitly defined in mise
* 1.13
* 1.25
* 1.63
* 1.75

On top of these explicitly defined line heights, there is a wide range of line
heights across our different pages. I believe we should consider assigning line
heights based on use cases (heading,body,etc.) rather than creating a list of
useable line heights.
