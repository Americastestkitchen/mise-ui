## For Developers

Breakpoint styling is located in the `./v2/src/styles/_tokens/_layout.scss` and is part of `mise.scss`.

### Using Breakpoints

Breakpoints can be used with the `mise.breakpoint()` function. This custom SASS function takes a `breakpoint` argument.

> **Note:** Media Queries should always set breakpoints with `min-width`.

#### SASS | *.scss

```
@use "./src/styles/mise.scss"

.example {
  @media only screen and (min-width: mise.breakpoint(sm)) {
    ...Styles
  }
}
```
