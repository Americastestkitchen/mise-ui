# Logos

## Usage
The `Logo` component takes a `type` prop and returns an svg. The dimensions
of the svg, as well accessibility features such as aria labels, should be
handled by whatever component implements the `Icon`.

```tsx filename="Logo.tsx"
// design-tokens/logos/Logo.tsx
type LogoProps = {
  type: LogoType;
};

const Logo = ({ type }: LogoProps) => {
  const LogoSvg = useLogoMap(type);
  return <LogoSvg />;
};

// How to pull Logo into another component
import Logo from 'design-tokens/logos/Logo';

<Logo type="americasTestKitchen" />
```
