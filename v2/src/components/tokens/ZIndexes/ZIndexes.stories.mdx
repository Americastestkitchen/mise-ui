# Z-Indexes

<br/>

## For Developers

Z-index styling is located in the `./v2/src/styles/_tokens/_layout.scss` and is part of `mise.scss`.

### Using Z-Indexes

Z-indexes can be used with the `mise.zIndex()` function. This custom SASS function takes a `level` and optional `adjustment` argument. `adjustment` can be anywhere between `1` and `99`, and allows for indexes to exist on the same level with a heirarchy, e.g. 101, 102. 103, etcâ€¦

#### SASS | *.scss

```
@use "./src/styles/mise.scss"

.example {
  z-index: mise.color(page); // returns 100

  &__adjusted {
    z-index: mise.color(page, 1); // returns 101
  }
}
```
