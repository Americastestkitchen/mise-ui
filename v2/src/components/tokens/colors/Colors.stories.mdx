### Using the Mise Color Palette

Colors can be used with the `MiseColor` function. This custom SASS function takes a `category` and optional `color`, and `variant` arguments.

> **Note:** All colors have the `variant`: `base`, the default color, and all categories have the `color`: `mid`, **excluding Editorial** colors which are numbered, starting with `color1`.

#### SASS | *.scss

```
// Import from `./v2/src/styles/styles.scss` or `./v2/src/styles/tokens.scss`
// Usage: MiseColor(category, color, variant)

.example {
  background-color: MiseColor(brand, mid, base);
  border-color: MiseColor(brand, dark, base);
  color: MiseColor(netural, white, base);

  &:hover {
    background-color: MiseColor(brand, mid, shade);
    border-color: MiseColor(brand, dark, shade);
  }
}
```
