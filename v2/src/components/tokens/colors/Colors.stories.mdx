## For Developers

### Using the Color Palette

Colors can be used with the `mise.color()` function. This custom SASS function takes a `hue` and optional `color`, and `variant` arguments. All colors should be implemented with `mise.color()`, **no hex codes, rgba, sass transforms, etcâ€¦** should be used directly in component styling. 

> **Note:** All colors have the `variant`: `base`, and all hues have the `color`: `mid`, **excluding Editorial** colors which are numbered, starting with `color1`.

#### SASS | *.scss

```
@use "./src/styles/mise.scss"

.example {
  background-color: mise.color(brand, mid, base);
  border-color: mise.color(brand, dark, base);
  color: mise.color(netural, white, base);

  &:hover {
    background-color: mise.color(brand, mid, shade);
    border-color: mise.color(brand, dark, shade);
  }
}
```
