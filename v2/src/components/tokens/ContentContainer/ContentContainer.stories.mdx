## For Developers

Content container styling is located in the `./v2/src/styles/_tokens/_layout.scss` and is part of `mise.scss`.

### Using the Content Container

The `ContentContainer` component is located in the `./v2/src/components/tokens/Layout/ContentContainer/ContentContainer.tsx`.

The component takes optional boolean props of `disableMaxWidth` and `disableGutters` to disable respective styling.

#### TSX | *.tsx

```
@import { ContentContainer } from "./mise/v2/src/components/Tokens/Layout/ContentContainer"

export const Example: React.FC = () => {
  return (
    <ContentContainer>
      ...Children
    </ContentContainer>
  )
};
```
