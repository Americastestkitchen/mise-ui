## For Developers

Spacing styling is located in the `./v2/src/styles/_tokens/_layout.scss` and is part of `mise.scss`.

### Using Spacing

Spacing can be used with the `mise.spacing()` function. This custom SASS function takes a `size` argument.

#### SASS | *.scss

```
@use "./src/styles/mise.scss"

.example {
  margin-top: mise.spacing(x1);
}
```
