# Card Types

Work in Progress, we will update/modify as we go. <br/>
Current design expectation is to reduce the number of props passed to a component and in passing too much logic.<br/>
Below is the documentation of current expected payload and design usage.

### Feed
```
<CardComponent card={payload.card} />
```
### Component
```
const CardComponent = ({ card }: CardComponentPropTypes) => {
  <div>
    <h1>{ card.title }</h1>
    <p>{ card.body }</p>
    {card.links.map(link => {
      return (
        <a href="{link.url}">{link.title}</a>
      )
    })}
    ....
  </div>
}
```
